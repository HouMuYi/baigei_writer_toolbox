# 白給的寫作工具箱

專為小說作家打造的一鍵化工具集。

## 專案簡介

作家創作小說或架空世界觀時，常需推算天體軌道、相對論效應、星際航行耗時與燃料，乃至極端天體的毀滅規模。本工具箱拆解各類繁瑣計算，提供獨立即時的網頁微工具，供創作者隨時調用。

## 工具清單

### 相對論、航行與宇宙學
- **相對論時間膨脹**：推算亞光速航行時的洛倫茲因子 γ，比對地球與船員時間流逝差異，兼算長度收縮與質量增長。
- **恆定加速度星際航行**：推算恆定 G 值加速度下，中點轉向減速時的船員時間、地球時間與航程最高速度。
- **宇宙學紅移**：輸入距離並選擇 Planck 或 HST 哈伯常數流派，推算河外星系退行速度與紅移 z。

### 天體、重力與軌道
- **洛希極限**：計算剛體與流體天體遭母星引力梯度撕裂的臨界軌道半徑。
- **希爾球引力範圍**：計算行星穩定約束衛星或太空站的引力主導半徑上限。
- **恆星適居帶**：輸入恆星光度，推算液態水與宜居生態存在的軌道內外邊界。
- **表面重力與逃逸速度**：輸入星球質量與半徑，推算表面 G 值、第一與第二宇宙速度，以及地表地平線視野極限。
- **金斯逸散大氣保留評估**：輸入星球溫度與逃逸速度，評估水、氧、氮、二氧化碳等氣體能否保留數十億年。
- **恆星絕對與視星等**：輸入恆星光度或絕對星等，推算特定光年距離下的夜空視星等與肉眼可見程度。
- **行星潮汐鎖定時間**：估算行星遭母星潮汐鎖定所需的演化時間，掌握永晝與永夜狀態。

### 能量與物理衝擊
- **動能撞擊能量**：計算亞光速或相對論高速物體撞擊時釋放的總焦耳能與 TNT 當量。
- **質能轉換能量**：計算反物質湮滅、氘氚核融合與核裂變的質能釋放能量。
- **旋轉太空站人工重力**：計算旋轉半徑與自轉 RPM 速度，並評估柯氏力動暈症與人體生理適應性。
- **黑洞物理與視界潮汐力**：計算黑洞史瓦西半徑與光球半徑，評估人體跨越事件視界時承受的潮汐力強度。
- **火箭方程與 Delta-v**：依齊奧爾科夫斯基火箭方程，計算目標速度變化量所需的推進劑質量與起飛濕重。

## 設計哲學與架構說明

本專案採無編譯輕量化與單頁動態模組架構：

- **單一通用 HTML 主模板 (`index.html`)**：全站共用一組 HTML 外殼，包含頁首導覽、主題切換與選單。免除重複維護樣式與 HTML 標籤的負擔。
- **獨立功能模組 (`tools/*.js`)**：各工具為獨立 JS 模組，自由主導中央容器繪製與計算邏輯，利於擴充互動元件。
- **零依賴與免編譯**：不引入 npm 與前端建置工具，兼顧零依賴輕量與單模板極簡維護需求。

## 功能特色

- **暗色與亮色護眼模式**：預設深色宇宙主題，可一鍵切換，保護寫作視力。
- **頁首無縫跨頁跳轉**：每頁頂部皆設有選單，可隨時返回總看板或切換工具。
- **設定方案備份與匯出**：可將常用天體或飛船參數存為預設方案，並匯出 JSON 檔備份或分享。
